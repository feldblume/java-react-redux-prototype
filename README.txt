## Backand

# Open Java project

- import project to Idea by build.gradle file
- java 1.8 required

# Start Java service

- ./gradlew build && java -jar build/libs/rest-service-proto-0.1.0.jar
- or buid by gradle task in the idea and run from Application class
- open http://172.17.88.128:8888/

## Frontend 

- cd frontend
- npm install react
- npm install react-dom
- npm install babel-core babel-loader babel-preset-es2015 babel-preset-react
- npm start
- open http://localhost:3333/
- push extract button
